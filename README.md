# Task Badge Board

Progetto d'esercitazione, per imparare e testare la tecnologia Next.js.

### Obiettivi del progetto

- Imparare a usare Next.js con App Router
- Gestire task e badge in modo reattivo
- Implementare logiche condizionali per sblocco badges
- Organizzare il codice in componenti riutilizzabili
- Visualizzare pagina dettagli per singolo task tramite rotte dinamiche

### Tecnologie utilizzate

- Next.js 16

- TypeScript

- CSS Custom Modules

- React Hooks

- pnpm

- Turbopack
